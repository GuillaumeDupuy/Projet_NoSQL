# Projet_NoSQL
Depot git du projet libre en NoSQL.

# Les étapes pour lancer le projet :

1. Cloner le projet 
2. Importer le dans Visual Studio
3. Faire les étapes Install et Configuration
4. Lancer le projet

# Install 

  - ```pip install pymongo```
  - ```pip install requests```
  - ```pip install pprint```
  - ```pip install json```
